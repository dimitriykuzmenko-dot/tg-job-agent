# Telegram Job Agent 💼

Бесплатный агент для поиска вакансий: мониторит публичные Telegram-каналы,
фильтрует посты по ключевым словам и присылает совпадения тебе в Telegram.

Работает на GitHub Actions — хостинг не нужен, стоит 0 рублей.

## Как это устроено

Каждые 30 минут GitHub Actions запускает `checker.py`. Скрипт читает
веб-версии каналов (`t.me/s/канал`), сравнивает посты с уже просмотренными
(`state.json`), фильтрует новые по ключевым словам из `config.json`
и шлёт совпадения через твоего бота.

## Настройка (10–15 минут)

### Шаг 1. Бот

Если бот уже есть (мы делали раньше) — используй его токен. Если нет:
напиши @BotFather в Telegram → `/newbot` → получи токен вида
`123456789:AAH...`.

**Важно:** напиши своему боту любое сообщение (например, «привет»),
иначе он не сможет писать тебе первым.

### Шаг 2. Узнай свой chat_id

Напиши боту @userinfobot — он пришлёт твой `Id` (число вида `123456789`).

### Шаг 3. Репозиторий

1. Создай **приватный** репозиторий на GitHub (например, `tg-job-agent`).
2. Загрузи в него все файлы проекта, сохранив структуру:
   ```
   checker.py
   config.json
   state.json
   README.md
   .github/workflows/job_agent.yml
   ```
   Проще всего через веб-интерфейс: Add file → Upload files.
   Папку `.github/workflows/` придётся создать через
   Add file → Create new file, указав путь
   `.github/workflows/job_agent.yml` прямо в имени файла.

### Шаг 4. Секреты

В репозитории: **Settings → Secrets and variables → Actions →
New repository secret**. Добавь два секрета:

| Name | Value |
|---|---|
| `TELEGRAM_BOT_TOKEN` | токен бота из шага 1 |
| `TELEGRAM_CHAT_ID` | твой id из шага 2 |
| `ANTHROPIC_API_KEY` | ключ API Anthropic для AI-фильтра (опционально) |

**Про AI-фильтр.** Если добавлен `ANTHROPIC_API_KEY`, каждый пост-кандидат
оценивает Claude Haiku по описанию твоего профиля из `config.json`
(блок `ai_filter.profile`) — он ловит вакансии, сформулированные без
точных ключевых слов («правая рука собственника», «забрать операционку»),
и отсекает мидл-позиции. Ключ создаётся на console.anthropic.com →
API Keys (нужно пополнить баланс минимум на $5; расход при твоих объёмах
~$1–3/мес). Без ключа агент работает по старой схеме — строгий список
`keywords`.

**Дедупликация.** Одна вакансия часто публикуется в нескольких каналах —
агент считает «отпечаток» текста и присылает только первую копию,
даже если ссылки и эмодзи в постах различаются.

### Шаг 5. Запуск

1. Вкладка **Actions** → если GitHub спросит, включи workflows.
2. Слева выбери **Job Agent** → справа **Run workflow** → Run.
3. Первый запуск только индексирует текущие посты (без уведомлений,
   чтобы не получить лавину старых вакансий). Уведомления пойдут
   со второго запуска — дальше всё работает само каждые 30 минут.

## Настройка под себя

Всё редактируется в `config.json` прямо на GitHub (карандаш → Commit):

- **channels** — список каналов (имя без `https://t.me/` и без `@`).
- **keywords** — слова-триггеры. Сравнение без учёта регистра, по вхождению
  подстроки: `"директор по маркетингу"` сработает и в «ищем директора
  по маркетингу» не сработает — поэтому в конфиге есть формы в разных
  падежах. Добавляй свои.
- **exclude_keywords** — если слово найдено, пост отбрасывается
  (отсекаем стажировки, ассистентов и т.п.).
- **max_post_age_hours** — посты старше этого возраста не присылаются.

## Если канал «молчит»

У некоторых каналов владельцы отключают веб-превью — тогда `t.me/s/канал`
не отдаёт посты, и скрипт пишет об этом в лог запуска (вкладка Actions →
последний запуск → Run checker). Такие каналы агент видеть не сможет —
их остаётся проверять вручную или через юзербота (другая архитектура).

## Лимиты и стоимость

- GitHub Actions: для **приватных** репозиториев бесплатно 2000 минут/мес.
  Один запуск занимает ~1 минуту → 48 запусков в день ≈ 1450 минут/мес —
  впритык, но влезает. Если захочешь запас — поменяй cron на `*/45` или
  сделай репозиторий публичным (для публичных лимита нет вообще,
  но конфиг с ключевыми словами будет виден всем; токены в Secrets
  в любом случае никому не видны).
- Telegram Bot API: бесплатно.

Итого: 0 ₽/мес.
